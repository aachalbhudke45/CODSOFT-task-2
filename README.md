# ⚡ EV Guard - Smart EV Battery Monitoring Landing Page

## 📌 Project Overview

EV Guard is a futuristic landing page designed to showcase a smart electric vehicle battery monitoring system. The project focuses on promoting EV safety through real-time battery monitoring, heat detection, intelligent alerts, and predictive protection features.

This project was developed as **Task 2 - Landing Page** for the **CODSOFT Web Development Internship**.

---

## 🚀 Features

- Modern and Responsive Design
- Futuristic EV-Themed User Interface
- Battery Health Monitoring Section
- Live Dashboard Cards
- Battery Scan Progress Animation
- Interactive Feature Cards
- Statistics Section
- Contact Form
- Mobile-Friendly Layout
- Hover Effects and Animations

---

- HTML5
- CSS3
- Google Fonts
- Font Awesome Icons
